# MatchaZuki Website - Setup Instructions

## Quick Start Guide

1. **Download placeholder images** or create your own:
   - Add logo.png (40x40px)
   - Add matcha-hero.jpg (hero background)
   - Add product images (ceremonial-matcha.jpg, culinary-matcha.jpg, matcha-set.jpg)

2. **Open the website**:
   - Double-click `index.html` to open in your browser
   - Or serve using a local server for full functionality

3. **Test responsive design**:
   - Resize your browser window
   - Use browser dev tools to test mobile view
   - Try the navigation and interactive elements
